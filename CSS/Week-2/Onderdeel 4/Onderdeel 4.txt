Onderdeel 4

1.	Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Het :: first-line pseudo-element wordt gebruikt om een speciale stijl toe te voegen aan de eerste regel van een tekst.

:: first-letter pseudo-element wordt gebruikt om een speciale stijl toe te voegen aan de eerste letter van een tekst.

:: before pseudo-element kan worden gebruikt om wat inhoud in te voegen vóór de inhoud van een element.

:: after pseudo-element kan worden gebruikt om wat inhoud in te voegen na de inhoud van een element.

2.	Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

:: selection pseudo-element komt overeen met het gedeelte van een element dat door een gebruiker is geselecteerd.

:: backdrop CSS pseudo-element is een box ter grootte van de viewport die direct wordt weergegeven onder elk element dat in volledig scherm wordt gepresenteerd.

:: grammar-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat de user-agent als grammaticaal incorrect heeft gemarkeerd.
